📡 Telegram: Backup tersimpan lokal
🔐 Cara Backup via Telegram:
1. Buka Telegram cari @BotFather
2. Ketik /newbot → ikuti petunjuk
3. Copy Bot Token (format: 123456:ABC-DEF)
4. Kirim pesan ke bot kamu lalu buka https://api.telegram.org/bot<TOKEN>/getUpdates
5. Copy angka chat_id (dari "chat":{"id":...})
6. Masukkan Token & Chat ID di bawah
💾 Data & Sharing
📱 Cara Sharing Data ke HP/User Lain:
1. Export data → file JSON tersimpan
2. Kirim file JSON via WhatsApp/Bluetooth
3. Di HP tujuan, login akun tujuan → Import file JSON

info Tentang Mughis Bank
Mughis Bank adalah sistem manajemen bisnis all-in-one yang dirancang untuk membantu pelaku usaha mengelola keuangan, pendataan, pengeluaran, pesanan, pelanggan, produk, serta piutang & hutang secara rapi, cepat, dan profesional.
